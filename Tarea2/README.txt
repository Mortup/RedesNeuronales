Se incluyen 3 programas:
1) bitGuesser.py: Algoritmo genetico para adivinar un numero binario.
2) wordGuesser.py: Algoritmo genetico para adivinar una palabra.
3) nQueen.py: Algoritmo genetico para encontrar la solucion al problema de nQueen.

Todos los programas se ejecutan con python 2.7 y no reciben parametros adicionales.
Las instrucciones se mostraran durante la ejecucion del archivo.

Ejemplo:
    $ python nQueen.py
    Largo del tablero: 
    $ 6
    Se ha encontrado l a combinacion luego de 845 generaciones.
    [(5, 3), (0, 2), (3, 4), (4, 0), (2, 1), (1, 5)]
    Imprimir tablero? (y/n)
    $ n

No se utiliza ninguna libreria externa.
Para un test rapido se sugiere no usar un n mayor que 12.
